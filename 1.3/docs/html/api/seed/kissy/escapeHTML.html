KISSY.escapeHTML()¶
String KISSY.escapeHTML (str)
将字符串经过 html 转义得到适合在页面中显示的内容, 例如替换 < 为 &lt;
Parameters:str (string) – 要显示在页面中的真实内容
Returns:经过 html 转义后的字符串
Return type:string

例如

KISSY.escapeHTML("<a>x</a>"); // =>  "&lt;a&gt;x&lt;/a&gt;"

Note

此函数只会对以下符号进行 escape：

& > < ` / " '
